## Java 集合相关（如下图）

![](img/collection.jpg)

`Java` 集合框架主要包括两种类型的容器   

一种是集合 `Collection`, 存储元素的集合, `Collection` 接口又有 3 种子类型, `List`、`Set` 和 `Queue`, 常用的有 `ArrayList`、`LinkedList`、`HashSet`、`LinkedHashSet`。       
一种是 `Map`, 存储键/值对映射, 常用的有 `HashMap`、`LinkedHashMap` 等等。
